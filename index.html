Gogl

search with google in command line

Download this project as a .zip file Download this project as a tar.gz file

Search Begin With CLI

Install

npm install gogl -g

Usage

  • Search With Google 
gogl someting
  • Search With So 
gogl so some words
  • Search With Baidu 
gogl bd some words
  • Also u can do like this:
gogl -e [so, baidu, google] some words

(If this tool can not used normally, Please open an issue to let me know!)
